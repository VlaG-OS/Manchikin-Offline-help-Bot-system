🎯 Тактика ботов

📊 Основная механика

Стадия A - Тип поведения (бросок 1d6):

  • 1-2 → Помогает игроку (33%)
  • 3-4 → Нейтральный (33%)
  • 5-6 → Помогает монстру (33%)

Стадия B - Сила влияния (бросок 1d6):

  • 1-2 → Слабое влияние (+2 бонус)
  • 3-4 → Среднее влияние (+4 бонус)
  • 5-6 → Сильное влияние (+6 бонус)

🎲 Нейтральное поведение (ОСЛАБЛЕННОЕ!)

Нейтральность теперь слабее, чтобы не доминировать:

  • Слабое: Всегда 0 (ничего не происходит)
  • Среднее: 33% = +1 игроку, 33% = +1 монстру, 33% = 0
  • Сильное: 50% = +1 игроку, 50% = +1 монстру

Нейтрал больше не даёт +2! Максимум ±1.

👑 Система уровней игроков (СБАЛАНСИРОВАНА!)

Определение лидера:

  • Лидер = игрок с самым высоким уровнем
  • Если уровни равны → "Никто не лидирует"

Если ходит лидер (МЯГКИЙ ШТРАФ):

  • Штраф применяется ТОЛЬКО если лидер впереди на 2+ уровня
  • 25% шанс: "Помощь игроку" → "Нейтральный"
  • Если разрыв 0-1 уровень → штрафа НЕТ (бой равный)

Если ходит не-лидер (ПЛАВНАЯ ПОМОЩЬ):

  • helpBoost = отставание × 5% (максимум 15%)
  • При отставании на 1 уровень: +5% помощи
  • При отставании на 2 уровня: +10% помощи
  • При отставании на 3+ уровня: +15% помощи (лимит!)
  • (35% + boost) шанс: "Помощь монстру" → "Нейтральный"/"Помощь игроку"
  • (25% + boost) шанс: "Нейтральный" → "Помощь игроку"

Если все игроки одного уровня:

  • Без модификаций, чистый рандом

💡 Итого

Боты создают хаос, но с умом: они балансируют игру, помогая слабым и мешая сильным. Это делает каждый бой непредсказуемым!

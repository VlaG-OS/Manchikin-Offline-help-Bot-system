# 🎲 Манчкин: Боты NPC

Веб-приложение для управления NPC-ботами в настольной игре Манчкин.

## 📋 Полное описание системы

### ✅ Что есть в системе

#### 1. **Панель управления игроками (слева)**
- Отображение всех игроков в игре (от 1 до 6)
- У каждого игрока свой **чистый уровень** (без шмоток)
- Кнопки **+/−** для изменения уровня игрока (от 1 до 10)
- Визуальные индикаторы:
  - **👑** — лидер (игрок с самым высоким уровнем)
  - **🎯** — игрок, который сейчас ходит
  - **Золотая рамка** — подсвечивает лидера
  - **Зеленая рамка** — подсвечивает текущего игрока
- Автоматическое определение лидера по уровню
- Если несколько игроков одного уровня = "Никто не лидирует"

#### 2. **Настройки боя (справа вверху)**
- **Количество ботов:** от 1 до 6
- **Кто сейчас ходит:** выбор активного игрока из списка
- Кнопка **"🎲 Бросить кости!"** для запуска механики ботов

#### 3. **Механика ботов (двухстадийная система)**

**Стадия A - Тип поведения (бросок 1d6):**
- 1-2 → 🟢 Помогает игроку (33%)
- 3-4 → 🟡 Нейтральный (33%)
- 5-6 → 🔴 Помогает монстру (33%)

**Стадия B - Сила влияния (бросок 1d6):**
- 1-2 → Слабое влияние (+2 бонус)
- 3-4 → Среднее влияние (+4 бонус)
- 5-6 → Сильное влияние (+6 бонус)

**Нейтральное поведение:**
- Слабое: 0 (ничего не происходит)
- Среднее: +1 случайно игроку или монстру (50/50)
- Сильное: +2 случайно игроку или монстру (50/50)

#### 4. **Система уровней и адаптивная логика**

Боты анализируют уровни игроков и принимают решения:

**Если ходит лидер (самый сильный игрок):**
- 30% шанс: "Помощь игроку" → "Нейтральный"
- Боты реже помогают доминирующему игроку

**Если ходит не-лидер (отстающий игрок):**
- 40% шанс: "Помощь монстру" → "Нейтральный" или "Помощь игроку"
- 30% шанс: "Нейтральный" → "Помощь игроку"
- Чем больше отставание по уровню, тем выше вероятность помощи (+10-20%)
- Боты чаще помогают слабым игрокам

**Если все игроки одного уровня:**
- Без модификаций, чистый рандом

#### 5. **Визуализация результатов**

**Карточки ботов (3 столбца):**
- Имя бота (🤖 Бот 1, 2, 3...)
- Два кубика: результаты бросков поведения и силы
- Поведение с цветовой маркировкой
- Сила влияния (слабое/среднее/сильное)
- Итоговый эффект

**Итоговая сводка:**
- Количество игроков в игре
- Кто лидирует (с уровнем) + кто ходит (с уровнем)
- Общий бонус игрокам
- Общий бонус монстру

#### 6. **Кнопка "📖 Как играют боты?"**
- Модальное окно с подробным объяснением всей механики
- Описание всех стадий
- Объяснение системы уровней
- Примеры работы логики

#### 7. **Адаптивный дизайн**
- **Десктоп:** 2 колонки (панель игроков + игровая область)
- **Планшет:** уменьшенная панель игроков
- **Телефон:** 1 колонка, все элементы друг под другом
- Карточки ботов адаптируются: 3 → 2 → 1 столбец

#### 8. **Дизайн**
- Средневековая тематика
- Пергаментные панели
- Золотые акценты
- Шрифты: MedievalSharp, Creepster
- Анимации: переворот карт, бросок кубиков
- Темный фоновый градиент с текстурой

### ❌ Чего НЕТ в системе

1. ❌ **Сохранение прогресса** — нет localStorage/базы данных
2. ❌ **История бросков** — не сохраняется история предыдущих ходов
3. ❌ **Звуковые эффекты** — нет звуков броска кубиков
4. ❌ **Многопользовательский режим** — только локальная игра
5. ❌ **Анимация уровней** — простое обновление без плавности
6. ❌ **Экспорт/импорт** — нельзя сохранить настройки в файл
7. ❌ **Статистика** — не ведется учет побед/поражений
8. ❌ **Темы оформления** — только одна тема
9. ❌ **Подсказки** — нет всплывающих подсказок для новичков
10. ❌ **Язык интерфейса** — только русский

### 🎯 Как работает вся система (пошагово)

1. **Игрок открывает сайт**
   - Видит 2 игроков по умолчанию (оба 1 уровня)
   - Может добавить до 6 игроков

2. **Настройка перед боем**
   - Игрок меняет уровни игроков кнопками +/−
   - Система автоматически определяет лидера
   - Выбирает, кто сейчас ходит
   - Выбирает количество ботов (1-6)

3. **Нажимает "Бросить кости!"**
   - Для каждого бота:
     - Бросок 1d6 для поведения
     - Система проверяет: лидер ходит или нет?
     - Применяет модификаторы (30-40% сдвиг вероятностей)
     - Бросок 1d6 для силы
     - Вычисляет финальный эффект

4. **Показываются результаты**
   - Карточки ботов с анимацией
   - Итоговая сводка с бонусами
   - Кнопка для просмотра тактики

5. **Применение результатов**
   - Игрок вручную прибавляет бонусы
   - Можно изменить уровни игроков
   - Выбрать следующего игрока
   - Повторить бросок для нового хода

### 🎲 Философия ботов

Боты НЕ играют случайно! Они создают **динамичный баланс**:
- Помогают слабым игрокам выживать
- Мешают сильным игрокам доминировать
- Добавляют хаос, но не убирают стратегию
- Делают каждый бой непредсказуемым
